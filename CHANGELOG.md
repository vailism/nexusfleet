# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-16

### Added
- Initial production release.
- Multi-tenant Organization architecture.
- Full Fleet Management (Vehicles, Fuel, Maintenance).
- Advanced Invoicing Engine with GST support.
- Stripe Subscription integration (Starter, Business, Enterprise).
- UploadThing file storage for receipts and documents.
- AI Insights Engine for predictive maintenance and revenue forecasting.
- Enterprise Audit Logging and RBAC.
- Sentry error monitoring and Winston structured logging.
- Optimized Docker production environment.
